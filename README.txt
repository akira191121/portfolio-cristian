PORTFOLIO DE CRISTIAN - INSTRUCCIONES RÁPIDAS

1. Abre la carpeta del proyecto.
2. Edita el archivo index.html.
3. Reemplaza estos enlaces:
   - LinkedIn
   - GitHub
   - Demo / Repositorios de proyectos
4. Si quieres poner tu CV:
   - crea una carpeta llamada /assets
   - guarda tu CV como cv-cristian.pdf
   - agrega un botón con href="assets/cv-cristian.pdf"

CÓMO SUBIRLO A VERCEL
1. Sube esta carpeta a GitHub.
2. Entra a Vercel.
3. Conecta tu cuenta de GitHub.
4. Importa el repositorio.
5. Deploy.

RECOMENDACIÓN
Antes de publicarlo, cambia los textos de proyectos por proyectos reales tuyos.
